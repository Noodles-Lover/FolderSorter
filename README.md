# FolderSorter - 多媒体文件分类工具

**在线体验：https://noodles-lover.github.io/FolderSorter/**

一个本地多媒体文件管理和分类工具，支持图片、音频、视频文件的快速预览和整理。暂不支持移动端。

## 🌟 特性

- **文件预览**：支持图片、音频、视频的实时预览
- **快速分类**：创建自定义分类文件夹，一键移动文件
- **文件管理**：重命名、删除、排序等基本文件操作
- **快捷键支持**：提高操作效率的键盘快捷键
- **本地化操作**：所有操作在本地进行，保护文件隐私
- **零安装**：直接在浏览器中运行，无需安装任何软件
- **个性化设置**：丰富的配置选项，支持保存用户偏好，下次打开自动恢复

## 🚀 使用方法

1. 访问在线版本 或 下载项目后用浏览器打开 `index.html` 文件
2. **首次使用建议先查看右上角 ❓ 的详细说明**
3. 点击「打开文件夹」选择要整理的文件夹
4. 创建分类文件夹并设置快捷键（如要进行分类）
5. 使用预览功能查看文件，通过分类按钮或快捷键进行整理

## ⌨️ 快捷键

- `Tab`：打开/关闭文件列表
- `←/→`：切换上一个/下一个文件
- `Delete`：删除当前文件
- 自定义分类快捷键：快速移动文件到对应分类

## 🛡️ 安全说明

- 本工具完全在本地运行，不联网，不会上传任何文件
- 删除操作默认移动到「_删除」文件夹，可在设置中改为永久删除（不进入回收桶）

## 🎯 适用场景

- 摄影师整理照片素材
- 视频创作者管理素材文件
- 软件录制文件的批量处理
- 项目的美术素材的整理

## 📋 系统要求

- 现代浏览器（支持 File System Access API）
- 建议使用 Chrome、Edge 等基于 Chromium 的浏览器

## 🔧 技术栈

- **前端框架**：Vue 3 (Options API)
- **UI组件库**：View UI Plus
- **样式**：原生 CSS
- **文件访问**：File System Access API
